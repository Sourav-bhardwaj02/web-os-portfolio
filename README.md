# 🚀 Getting Started

## Clone Repository

```bash
git clone https://github.com/Sourav-bhardwaj02/web-os-portfolio.git
```

## Navigate

```bash
cd web-os-portfolio/frontend
```

## Install Dependencies

```bash
npm install
```

## Run Development Server

```bash
npm run dev
```

Open

```
http://localhost:3000
```
